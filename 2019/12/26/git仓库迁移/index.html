码云 Git 仓库迁移到 git.jnaw.top 服务器

公共账户

  • 账户：git
  • 密码：git

免密登录

  • 先把自己电脑的公钥添加到 git 账户中
  • ~/.ssh/authorized_keys 【追加到此文件中】

以 ncp 项目为例迁移

  • 先在新的git服务器上创建一个空的项目ncp

克隆旧的仓库，推送到新的服务器

  • 在本地 Git Bash 中执行命令
  • git@gitee.com:allview-cwbu/ncp.git 为原仓库地址
  • 会生成一个文件夹：ncp.git
  • 进入文件夹中
  • 推送到新的服务器
  • git@git.jnaw.top:cwbu/ncp 为新的仓库地址
完
